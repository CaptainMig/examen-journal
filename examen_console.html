NERVA Kernel — State Classification Logic
// ── INPUTS
entropy ∈ [0,1] // von Neumann analog — uncertainty of the decision state
stakes ∈ [0,1] // consequence magnitude
evidence ∈ [-1,1] // signed evidence balance (for − against)
owd ∈ {0,1} // one-way door: irreversibility flag
// ── KERNEL DERIVATION
posture = (evidence + 1) / 2 // normalize evidence to [0,1]
confidence = posture × (1 − entropy) // certainty-weighted conviction
// ── STATE CLASSIFICATION
if owd AND confidence < 0.70 AND stakes > 0.5
  → CONSULT // One-Way Brake
elif confidence ≥ 0.72 AND entropy < 0.45
  → COMMIT
elif stakes < 0.25 AND entropy > 0.55
  → WAIT
elif confidence < 0.30 AND stakes > 0.65
  → TOXIC
elif confidence ≥ 0.48
  → HOLD
else
  → CONSULT
// ── REBOUND / CALIBRATION
lift = follow_hit_rate − override_hit_rate // falsification metric
lift > +0.20 → NERVA adds signal
lift ∈ [−0.10, +0.10] → mirrors gut only
lift < −0.10 → NERVA degrades judgment
// ── DENSITY MATRIX ANALOG
ρ = Σᵢ pᵢ |ψᵢ⟩⟨ψᵢ|  // mixed decision state
S(ρ) = −Tr(ρ log ρ) // von Neumann entropy; maps to entropy input
Bloch vector r = (⟨σₓ⟩, ⟨σᵧ⟩, ⟨σᵤ⟩)
|r| → 1: pure state (COMMIT or TOXIC)
|r| → 0: mixed state (WAIT or CONSULT)
Epistemic Disclosure

The NERVA kernel is a heuristic scoring model, not a derivation from quantum information axioms. The density matrix and Bloch sphere provide a rigorous mathematical vocabulary for decision uncertainty — entropy, superposition, mixed states — but the specific thresholds (0.72, 0.48, etc.) were calibrated by judgment, not measurement. The kernel is the scale's needle (examen), not the weight of the objects on the pan.

The falsification criterion — lift > +0.20 over ≥10 scored outcomes with ≥3 overrides — is pre-committed and immovable. That is the science.
The Five States — Operational Definitions
COMMIT   Act. Evidence leans clearly for, entropy is manageable, confidence is high. The needle tips.
HOLD     Keep position. Defensible but not clear enough. Maintain without advancing. Watch for the entropy-reducing signal.
WAIT     Stand down. Low stakes and high noise: the cost of waiting is lower than the cost of being wrong.
CONSULT  Seek counsel — your wife, your advisor, your coach, your confessor, the data you don't have yet.
TOXIC    Walk away. Bad regardless of upside. High stakes, low confidence, evidence against. Expected value is negative.
